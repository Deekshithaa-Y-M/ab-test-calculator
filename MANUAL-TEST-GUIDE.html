Manual Testing Guide Ready to Test

Note: This guide helps you manually verify all functionality of the A/B Test Significance Calculator. Open the application in a separate tab/window and follow the test cases below.
🚀 Open A/B Test Calculator

1. Basic Functionality Tests

Test 1.1: Valid Calculation

Steps:
  1. Enter Control: 1000 visitors, 100 conversions
  2. Enter Variant: 1000 visitors, 150 conversions
  3. Click "Calculate Significance"

Expected Results:

Control rate shows 10.0%
Variant rate shows 15.0%
Confidence level is displayed (should be high, >95%)
Green badge shows "Significant Winner!"

Test 1.2: Not Significant Result

Steps:
  1. Enter Control: 100 visitors, 10 conversions
  2. Enter Variant: 100 visitors, 12 conversions
  3. Click "Calculate Significance"

Expected Results:

Control rate shows 10.0%
Variant rate shows 12.0%
Confidence level is low (≤95%)
Gray badge shows "Not Significant yet"

Test 5.2: Mobile View (< 640px)

6. Accessibility Tests

Test 6.1: Keyboard Navigation

Steps:
  1. Click in the browser address bar
  2. Press Tab repeatedly to navigate through the page
  3. Use Enter to activate the calculate button

Expected Results:

All input fields are reachable via Tab
Calculate button is reachable via Tab
Focus indicator is visible (blue outline)
Tab order is logical
Enter key activates calculate button
No keyboard traps

Test 6.2: Screen Reader (Optional - requires screen reader software)

Steps:
  1. Enable screen reader (NVDA on Windows, VoiceOver on Mac)
  2. Navigate through the page
  3. Listen to announcements

Expected Results:

Page title is announced
Input labels are read correctly
Button purpose is clear
Results are announced when displayed
